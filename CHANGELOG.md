# Changelog

## [1.0.0] - 2026-06-18

### iOS アプリ（Flutter）— 初回リリース

**トレーニング**
- 種目の追加・削除・並び替え（ドラッグハンドル式）
- トレーニング記録（種目・セット数・重量・回数）
- セット追加時に前セットの重量を引き継ぎ
- 種目をカードから切り替え可能
- 自己ベスト表示
- ルーティン（定型メニュー）の作成・編集・並び替え・ワンタップ開始
- トレーニング履歴の表示・削除

**食事**
- 食事記録（カロリー・タンパク質・脂質・炭水化物・メモ）
- 1日の PFC 合計・カロリー集計

**体重**
- 体重・体脂肪率・骨格筋量の記録
- 推移グラフ（折れ線）

**その他**
- ダーク／ライト／システム テーマ切り替え（サーバー永続化）
- プロフィール画像アップロード
- アカウント登録・ログイン・ログアウト
- Firebase Crashlytics + Analytics

### Web（Next.js）

- プライバシーポリシーページ追加（`/privacy`）
- サポートページ追加（`/support`）
